# varpal-images

Banco de assets gráficos de **VARPAL Group Corporation** (VP CORP · Import & Export) para
publicación en Instagram / Facebook. Courier y freight forwarding **Miami → Venezuela**.

## Estructura

```
assets/
  instagram/      Posts de feed 1:1 (1080x1080 / 1254x1254)
  stories/        Stories 9:16 (pendiente)
  reels-covers/   Portadas de Reels 9:16 (pendiente)
legal/
  privacy.html    Política de privacidad (Meta App Review)
docs/
  BRAND_GUIDELINES.md            Línea gráfica oficial VP CORP
  EDITORIAL_CALENDAR.md          Calendario editorial + captions
  INSTAGRAM_PUBLISHING_STATUS.md Estado de publicación nativa Meta
```
